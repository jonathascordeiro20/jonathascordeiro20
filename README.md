## 🔭 O que estou construindo

- **[OpenInfra Logger](https://github.com/jonathascordeiro20/openinfra-logger)** —
  Logging estruturado open source para Node, Python, Go e Rust. Zero dependências,
  auto-redação LGPD/GDPR, OpenTelemetry nativo e análise de logs por IA (local-first).
- **[APOF Engine](https://github.com/jonathascordeiro20/apof-engine)** —
  Framework open source de governança autônoma para times de produto de IA.
  Arquitetura Dual-Track com consenso multiagente (ML + FinOps), instalável como
  skill em Claude Code, Cursor, Copilot e Windsurf.
  Whitepaper publicado — DOI: [10.5281/zenodo.20370494](https://doi.org/10.5281/zenodo.20370494)
- **Nexus Hub** — Suíte de extensões para Azure DevOps:
  qualidade de backlog com IA (framework INVEST), forecasting Monte Carlo com
  ajuste semântico de risco e automação visual de work items.
- Produtos para o ecossistema **Atlassian** (Forge + Rovo) focados em flow
  metrics e previsibilidade de entrega. 👀
